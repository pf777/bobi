# 把本项目推到 GitHub 仓库 bobi

仓库地址：https://github.com/pf777/bobi

在 **PowerShell** 或 **CMD** 里依次执行（在 `d:\Android` 目录下）：

```powershell
cd d:\Android

git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/pf777/bobi.git
git push -u origin main
```

执行完后，代码就在 GitHub 上了，可以去 **Actions** 里等自动构建，然后下载 APK。

如果执行 `git push` 时提示要登录，按提示用浏览器或 Personal Access Token 完成认证即可。
