# 把本项目推到 GitHub 仓库 bobi

仓库地址：https://github.com/pf777/bobi

在 **PowerShell** 或 **CMD** 里依次执行（在 `d:\Android` 目录下）：

```powershell
cd d:\Android

git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/pf777/bobi.git
git push -u origin main
```

执行完后，代码就在 GitHub 上了，可以去 **Actions** 里等自动构建，然后下载 APK。

如果执行 `git push` 时提示要登录，按提示用浏览器或 Personal Access Token 完成认证即可。

---

### 若出现「git-credential-manager.exe - 系统错误 / 字符串绑定无效」

说明 GitHub 在浏览器里已连上，但本机凭据管理器报错。可以：

1. **先看是否已经推送成功**：打开 https://github.com/pf777/bobi 看是否有你的代码；有的话可忽略弹窗，直接去 Actions 下载 APK。
2. **若没成功，用 Token 绕过凭据管理器**：
   - GitHub 网页 → 头像 → **Settings** → **Developer settings** → **Personal access tokens** → **Generate new token**，勾选 `repo`，复制生成的 token。
   - 在项目目录执行：
   ```powershell
   git remote set-url origin https://你的token@github.com/pf777/bobi.git
   git push -u origin main
   ```
   - 推完后改回地址（避免 token 留在配置里）：
   ```powershell
   git remote set-url origin https://github.com/pf777/bobi.git
   ```
3. **或更新 Git**：从 https://git-scm.com/download/win 安装最新版 Git 后重试。
